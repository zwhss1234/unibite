# UniBite — Οδηγίες Εγκατάστασης από Μηδέν

---

## Τι θα χρειαστείς

- Υπολογιστή με **Windows 10 ή 11**
- Σύνδεση στο internet (μόνο για την εγκατάσταση του XAMPP)
- Τα αρχεία του project (ο φάκελος `Unibite`)

---

## Βήμα 1 — Εγκατάσταση XAMPP

1. Πήγαινε στη διεύθυνση: **https://www.apachefriends.org**
2. Κατέβασε την έκδοση για **Windows** (το κουμπί "XAMPP for Windows")
3. Τρέξε το αρχείο που κατέβηκε (`xampp-windows-x64-...-installer.exe`)
4. Στην εγκατάσταση:
   - Άφησε τα πάντα **ως έχουν** (default options)
   - Ο φάκελος εγκατάστασης πρέπει να είναι `C:\xampp`
   - Πάτα **Next** σε όλα → **Finish**
5. Στο τέλος θα ανοίξει αυτόματα το **XAMPP Control Panel**

> Αν ρωτήσει το Windows Defender Firewall, πάτα **"Allow access"**

---

## Βήμα 2 — Τοποθέτηση των αρχείων

1. Αντίγραψε τον φάκελο **`Unibite`** στο:
   ```
   C:\xampp\htdocs\
   ```
   Έτσι ώστε να υπάρχει ο φάκελος:
   ```
   C:\xampp\htdocs\Unibite\
   ```

2. Βεβαιώσου ότι μέσα υπάρχουν τα αρχεία:
   ```
   C:\xampp\htdocs\Unibite\
   ├── index.html
   ├── style.css
   ├── script.js
   ├── refresh.bat
   └── backend\
       ├── config.php
       ├── auth.php
       ├── ads.php
       ├── requests.php
       ├── stats.php
       ├── unibite.sql
       └── seed_data.sql
   ```

---

## Βήμα 3 — Εκκίνηση του XAMPP

1. Άνοιξε το **XAMPP Control Panel**  
   (αν το έκλεισες: Start Menu → αναζήτησε "XAMPP")

2. Πάτα **Start** δίπλα στο **Apache**

3. Πάτα **Start** δίπλα στο **MySQL**

4. Περίμενε μερικά δευτερόλεπτα — και οι δύο πρέπει να γίνουν **πράσινοι** με κείμενο "Running"

   ![XAMPP Running](https://i.imgur.com/example.png)

> ⚠️ Αν το **MySQL δεν ξεκινά** (μένει κόκκινο), δες το Βήμα 7 παρακάτω.

---

## Βήμα 4 — Δημιουργία της Βάσης Δεδομένων

Αυτό γίνεται **μόνο μία φορά**.

1. Πάτα **Win + R**, γράψε `cmd` και πάτα Enter
2. Στο παράθυρο που άνοιξε, γράψε **ακριβώς** αυτές τις εντολές:

```
chcp 65001
```
*(πάτα Enter)*

```
C:\xampp\mysql\bin\mysql.exe -u root < "C:\xampp\htdocs\Unibite\backend\unibite.sql"
```
*(πάτα Enter)*

3. Αν δεν εμφανιστεί κανένα error, η βάση δημιουργήθηκε επιτυχώς ✅

> Αν δεις μήνυμα `ERROR 1045` → δες το Βήμα 7.

---

## Βήμα 5 — Άνοιγμα της Εφαρμογής

1. Άνοιξε τον **browser** σου (Chrome, Firefox, Edge)
2. Στη γραμμή διεύθυνσης γράψε:

```
http://localhost/Unibite/
```

3. Πρέπει να εμφανιστεί η σελίδα του **UniBite** με φόρμα εγγραφής

### Εγγραφή (νέος χρήστης)
1. Επίλεξε **"Εγγραφή"**
2. Βάλε ένα username, το email σου, και επίλεξε ρόλο:

## Βήμα 7 — Αντιμετώπιση Προβλημάτων

### Το MySQL δεν ξεκινά στο XAMPP
Πιθανόν να τρέχει ήδη κάποια άλλη MySQL στον υπολογιστή σου.

**Λύση:**
1. Άνοιξε το XAMPP Control Panel
2. Δίπλα στο MySQL πάτα **Config** → **my.ini**
3. Βρες τη γραμμή `port=3306` και άλλαξέ την σε `port=3307`
4. Αποθήκευσε και ξανακίνησε το MySQL
5. Άνοιξε το αρχείο `C:\xampp\htdocs\Unibite\backend\config.php`
6. Βρες τη γραμμή `$port = 3306;` και άλλαξέ την σε `$port = 3307;`

---

### Το site εμφανίζει "Σφάλμα σύνδεσης με τον server"
- Βεβαιώσου ότι Apache **και** MySQL τρέχουν (πράσινα) στο XAMPP
- Βεβαιώσου ότι έτρεξες την εντολή του Βήματος 4

---

### Ανοίγω το site αλλά βλέπω λευκή σελίδα ή PHP code
- Βεβαιώσου ότι ο φάκελος είναι στο `C:\xampp\htdocs\Unibite\` (όχι αλλού)
- Βεβαιώσου ότι ο Apache τρέχει

---

### Ξέχασα να τρέξω την εντολή του Βήματος 4
Αν το site εμφανίζεται αλλά βγάζει σφάλμα στη σύνδεση/εγγραφή, σημαίνει ότι η βάση δεδομένων δεν υπάρχει. Τρέξε ξανά την εντολή του **Βήματος 4**.

---

## Βήμα 6 — Ανανέωση Δεδομένων (για επόμενες φορές)

Οι αγγελίες φαγητού λήγουν αυτόματα μετά από **48 ώρες**. Αν ανοίξεις το site και το feed είναι άδειο, κάνε διπλό κλικ στο αρχείο:

```
C:\xampp\htdocs\Unibite\refresh.bat
```

Αυτό προσθέτει αυτόματα 8 φρέσκες αγγελίες και επαναφέρει τα credits των test χρηστών.

> Δεν χρειάζεται cmd ή τίποτα άλλο — απλώς διπλό κλικ.

## Σύνοψη (για γρήγορη αναφορά)

```
1. Εγκατάσταση XAMPP από apachefriends.org
2. Αντιγραφή φακέλου Unibite → C:\xampp\htdocs\Unibite\
3. XAMPP Control Panel → Start Apache + Start MySQL
4. cmd → chcp 65001 → εντολή mysql για import του unibite.sql
5. Browser → http://localhost/Unibite/
6. Άδειο feed; → διπλό κλικ στο refresh.bat
```
