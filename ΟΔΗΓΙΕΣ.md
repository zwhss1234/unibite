# UniBite — Οδηγίες Εκτέλεσης

## Τι χρειάζεσαι

- **XAMPP** (εγκατεστημένο στο `C:\xampp`)  
  Περιέχει: Apache 2.4, MariaDB 10.4, PHP 8.2

---

## Βήμα 1 — Εκκίνηση XAMPP

1. Άνοιξε το **XAMPP Control Panel** (από το Start Menu ή από `C:\xampp\xampp-control.exe`)
2. Πάτα **Start** δίπλα στο **Apache**
3. Πάτα **Start** δίπλα στο **MySQL**
4. Και οι δύο πρέπει να γίνουν **πράσινοι**

> ⚠️ **Σημαντικό για αυτόν τον υπολογιστή:** Υπάρχουν δύο MySQL instances.  
> Το XAMPP MySQL τρέχει στη **θύρα 3307** (όχι στη 3306).  
> Πρέπει να αλλάξεις το `backend/config.php` ώστε `$port = 3307;`

---

## Βήμα 2 — Πρώτη φορά: Εισαγωγή Βάσης Δεδομένων

> Αν έχεις ήδη τρέξει αυτό, **παράλειψε το βήμα** — η βάση παραμένει ακόμα και αν κλείσεις τον υπολογιστή.

Άνοιξε **Command Prompt** (cmd) ως κανονικός χρήστης και τρέξε:

```
chcp 65001
C:\xampp\mysql\bin\mysql.exe -u root -P 3307 --default-character-set=utf8mb4 < "C:\Users\User\Unibite\backend\unibite.sql"
```

Αυτό δημιουργεί τη βάση `unibite_db` με:
- 3 πίνακες (users, ads, requests)
- 6 test χρήστες (4 μάγειρες, 2 καταναλωτές)
- 5 δείγματα φαγητών
- Views, Stored Procedures

---

## Βήμα 3 — Άνοιγμα της Εφαρμογής

Άνοιξε τον browser (Chrome / Firefox) και πήγαινε στη διεύθυνση:

```
http://localhost/unibite/
```

> ❌ **ΜΗΝ** ανοίγεις το αρχείο `index.html` απευθείας (double-click).  
> Το `file://` δεν εκτελεί PHP — η εφαρμογή δεν θα δουλέψει.

## Πώς λειτουργεί η εφαρμογή

### Ως Καταναλωτής
1. **Feed** → βλέπεις τα διαθέσιμα φαγητά (τελευταίες 48 ώρες)
2. Πάτα **"Παραγγελία"** → επίλεξε μερίδες → **"Παραγγελία"**
3. Τα credits σου αφαιρούνται αμέσως
4. Πήγαινε στην καρτέλα **"Αιτήματα"** → βλέπεις την κατάσταση
5. Μόλις εγκριθεί από τον μάγειρα → **βαθμολόγησε** με 1-5 ⭐
6. Η βαθμολογία πληρώνει τον μάγειρα (>3 ⭐ = +1 bonus credit)

### Ως Μάγειρας
1. **Αγγελίες** → δημιούργησε νέα αγγελία φαγητού
2. **Αιτήματα** → βλέπεις ποιος παράγγειλε → **Αποδοχή** ή **Άρνηση**
3. Αν αποδεχτείς, τα credits σου μπαίνουν μόλις ο καταναλωτής βαθμολογήσει

### Σύστημα Credits
| Ενέργεια | Credits |
|---|---|
| Εγγραφή | +5 🪙 |
| Παραγγελία | −(κόστος × μερίδες) 🪙 |
| Μάγειρας παραλαβή (rating ≤ 3) | +(κόστος × μερίδες) 🪙 |
| Μάγειρας παραλαβή (rating > 3) | +(κόστος + 1) × μερίδες 🪙 |
| No-show ποινή | −1 🪙 |

## Αντιμετώπιση Προβλημάτων

### "Σφάλμα σύνδεσης με τον server"
- Βεβαιώσου ότι ο Apache και η MySQL τρέχουν στο XAMPP Control Panel
- Βεβαιώσου ότι ανοίγεις `http://localhost/unibite/` και όχι `file://`

### Η σελίδα δεν φορτώνει καθόλου
- Έλεγξε αν ο Apache τρέχει (πράσινος στο XAMPP)
- Δοκίμασε `http://localhost/` — αν εμφανίζεται η σελίδα XAMPP, το Apache δουλεύει

### Χάθηκαν τα δεδομένα
- Τα δεδομένα αποθηκεύονται στη MySQL και παραμένουν ακόμα και μετά από επανεκκίνηση
- Αν τρέξεις ξανά το `unibite.sql`, **δεν** διαγράφει τα υπάρχοντα δεδομένα (το DROP DATABASE έχει αφαιρεθεί)

### Ξέχασα αν τρέχει η MySQL
Άνοιξε Command Prompt και τρέξε:
```
netstat -ano | findstr :3307
```
Αν εμφανιστεί αποτέλεσμα, η MySQL τρέχει.
